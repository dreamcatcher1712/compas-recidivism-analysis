# COMPAS Recidivism Analysis

Analyzing racial bias in algorithmic risk assessment for Black vs White women.

## Key Findings
- Nearly identical recidivism rates: Black (37.0%) vs White (35.3%)
- COMPAS assigns significantly different risk scores despite similar outcomes
- No statistical difference in time to recidivism (p = 0.45)
